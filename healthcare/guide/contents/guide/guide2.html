기본규약

  • style.css 파일을 기본으로 하며, 한 파일에 CSS 내용을 담는다
  • 요소는 주석으로 분리해 가독성을 높이고 의미를 부여한다.
  • 개발후의 기본적인 수정은 /* '130101 list type1 modify */ 형식을 타이틀로 삼으며, 한글은 사용하지 않는다.
  • 핵의 사용은 지양하지만, cross browsing을 우선적으로 생각한다.
  • 셀렉터 {속성: 수치; 속성: 수치;} 형식으로 표기한다.
